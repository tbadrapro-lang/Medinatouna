# Guide d'utilisation — Tableau de bord Les Bons Plans d'Arabie

Ce guide explique simplement comment utiliser votre tableau de bord (dashboard) pour gérer votre site.

## 1. Se connecter au dashboard

1. Allez sur `https://lesbonsplansdarabie.com/admin/login`
2. Entrez votre mot de passe administrateur
3. Vous arrivez sur le tableau de bord avec le menu : Contenus, Mes clients, Relances email, Réglages

Votre session reste active 7 jours, ensuite il faudra vous reconnecter.

## 2. Modifier un prix de pack (Institut, Camp bédouin, E-books)

1. Cliquez sur **Contenus** dans le menu
2. Choisissez la catégorie (Packs Institut, Packs Camp bédouin, E-books...)
3. Cliquez sur le pack que vous voulez modifier
4. Changez le prix (et les autres infos si besoin)
5. Cliquez sur **Enregistrer**

Le changement est visible sur le site immédiatement (ou dans la minute qui suit).

## 3. Ajouter un bon plan avec photo

1. Cliquez sur **Contenus**
2. Choisissez la catégorie correspondante (ex : Bons plans)
3. Cliquez sur **Ajouter**
4. Remplissez le titre, la description, le prix si besoin
5. Cliquez sur la zone photo pour téléverser une image depuis votre téléphone ou ordinateur
6. Cochez **Actif** pour que le bon plan apparaisse sur le site
7. Cliquez sur **Enregistrer**

Astuce : vous pouvez réorganiser l'ordre d'affichage avec le champ "Ordre" (les plus petits numéros apparaissent en premier).

## 4. Suivre vos clients et changer leur statut

1. Cliquez sur **Mes clients**
2. Vous voyez la liste de toutes les personnes qui ont rempli un formulaire sur le site
3. Utilisez les filtres en haut pour chercher par service, par statut, ou par nom/email
4. Chaque client a un badge de statut cliquable :
   - **Nouveau** (bleu) → la personne vient d'arriver
   - **Contacté** (or) → vous l'avez déjà contactée
   - **Converti** (vert) → elle est devenue cliente
   - **Perdu** (gris) → elle ne donne plus suite
5. Cliquez sur le badge pour passer au statut suivant
6. Cliquez sur un client pour ouvrir sa fiche détaillée : vous pouvez y ajouter des notes (sauvegardées automatiquement) et le contacter en un clic via **WhatsApp** ou **Email**
7. Le bouton **⬇️ Exporter CSV** en haut télécharge la liste complète de vos clients dans un fichier Excel

## 5. Envoyer une relance email

1. Cliquez sur **Relances email**
2. Choisissez un modèle de message parmi les 3 proposés (Relance douce, Dernières places, Nouveauté), vous pouvez modifier le texte et l'objet
3. Sélectionnez les destinataires avec les filtres (service, statut)
4. Vérifiez l'aperçu en bas
5. Cliquez sur **Envoyer à X personne(s)** puis confirmez

⚠️ Pour éviter de spammer vos clients, une même personne ne peut pas recevoir plus d'un email tous les 3 jours.

## 6. Réglages du site

Dans **Réglages**, vous pouvez modifier sans toucher au code :
- Le numéro WhatsApp principal et celui du camp bédouin
- L'email de contact
- Les liens Instagram, TikTok et PayPal

Tapez la nouvelle valeur puis cliquez sur **Enregistrer**. Le site utilise automatiquement ces nouvelles informations.

## 7. En cas de problème

- Pour toute question sur le contenu, les prix ou les clients : contactez **Badra**
- Pour tout problème technique (site qui ne fonctionne pas, bug, dashboard inaccessible) : contactez **Proxia IA**
